# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/34091325-2dd8-482b-8537-2e281d7b76e4

## Project structure

- `frontend/` contains the React app, pages, components, styles, and static assets.
- `backend/` contains the Express server, routes, controllers, models, middleware, and seed data.
- `dist/` is the production frontend build output served by the backend.

## Run locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the required environment variables in your local env file
3. Run the app:
   `npm run dev`
